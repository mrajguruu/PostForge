# Screenshots

This directory contains screenshots of the PostForge application.

## How to Add Screenshots

1. **Take screenshots** of your running application:
   - Admin Dashboard (`dashboard.png`)
   - Post Management (`posts.png`)
   - Login Page (`login.png`)
   - Public Homepage (`homepage.png`)
   - Profile Settings (`settings.png`)

2. **Recommended tool:** Use Snipping Tool (Windows), Screenshot (Mac), or browser DevTools

3. **Image guidelines:**
   - Format: PNG (preferred) or JPG
   - Resolution: 1920x1080 or similar
   - File size: < 500KB each
   - Show real data (use the demo posts/categories)

4. **Save files** in this directory with exact names:
   - `dashboard.png`
   - `posts.png`
   - `login.png`
   - `homepage.png`
   - `settings.png`

## Quick Screenshot Checklist

When running the application, capture:

- [ ] **Dashboard** - Show statistics, recent posts, and recent comments
- [ ] **Posts** - Show the posts table with multiple entries
- [ ] **Login** - Show the login page with demo credentials visible
- [ ] **Homepage** - Show featured post and grid layout with posts
- [ ] **Settings** - Show profile picture upload and password change

## Tips

- Use a consistent browser window size
- Clear browser console/devtools before capturing
- Show actual data (not empty states)
- Capture full screen or centered content
- Use light theme for better visibility
